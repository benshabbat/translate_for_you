# 👨‍🏫 מדריך למורה

ברוכים הבאים! מדריך זה מיועד לך, המורה, להבין ולהשתמש באפליקציה ללימוד אנגלית.

---

## 📋 סקירה כללית

### מה האפליקציה עושה?
אפליקציה זו מאפשרת לתלמידים:
- ✅ ליצור חשבון אישי
- ✅ לבנות מאגר מילים אנגלי-עברי משלהם
- ✅ לתרגל את המילים באמצעות חידונים
- ✅ לעקוב אחר ההתקדמות שלהם

### למה זה שימושי?
- **למידה אישית**: כל תלמיד יכול להתמקד במילים שקשות לו
- **מעקב**: התלמידים רואים את האחוז הצלחה שלהם
- **עצמאות**: התלמידים מנהלים את הלמידה שלהם
- **נגישות**: זמין מכל מכשיר עם דפדפן

---

## 🚀 התחלה מהירה

### שלב 1: הכנה
1. ודא ש-Node.js מותקן (מהדורה 18+)
2. התקן MongoDB או השתמש ב-MongoDB Atlas (חינם)
3. פתח את הפרויקט ב-VS Code

### שלב 2: הפעלת הפרויקט
**דרך קלה:**
- הרץ את `start.bat` (Windows)
- או `start.ps1` (PowerShell)

**דרך ידנית:**
1. פתח 2 טרמינלים
2. בטרמינל 1: `cd server && npm run dev`
3. בטרמינל 2: `cd client && npm run dev`

### שלב 3: צור משתמש דמו
1. היכנס ל-http://localhost:3000
2. לחץ "הירשם"
3. צור משתמש: `teacher` / סיסמה: `123456`
4. התחל להוסיף מילים לדוגמה

## 📚 העבודה עם התלמידים

### שיעור ראשון - היכרות
1. הצג את האפליקציה למחלקה
2. הסבר את הרעיון: מאגר אישי של מילים
3. הראה איך מוסיפים מילה
4. הראה איך מתרגלים

### המלצות לשימוש

**שיטה 1 - רשימת מילים משותפת:**
- תן למחלקה רשימת 20 מילים
- כל תלמיד יכניס אותן למערכת שלו
- תבקש מהם לתרגל בבית

**שיטה 2 - למידה אישית:**
- כל תלמיד מוסיף מילים שהוא נתקל בהן
- מספרים, תחביבים, מילים שקשות להם
- תבקש להוסיף 5 מילים חדשות בשבוע

**שיטה 3 - תחרות:**
- קבע יעד: 50 מילים + 80% הצלחה
- עקוב אחר ההתקדמות
- תן פרסים לתלמידים המצטיינים

### מעקב אחר התקדמות
כרגע, כל תלמיד רואה את הסטטיסטיקות שלו:
- מספר מילים כולל
- מילים שתורגלו
- אחוז הצלחה

💡 **עתידי:** אפשר להוסיף פאנל למורה שרואה את כל התלמידים

## 📊 נתונים וסטטיסטיקות

### מה התלמיד רואה?
- **סה"כ מילים**: כמה מילים יש לו במאגר

### מה נשמר?

- תאריך תרגול אחרון
