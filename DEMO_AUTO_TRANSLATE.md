# 🎬 הדגמת תרגום אוטומטי

## איך לבדוק שהתכונה עובדת?

### שלב 1: וודא שהשרת והקליינט פועלים
```powershell
# טרמינל 1 - Backend
cd server
npm run dev

# טרמינל 2 - Frontend  
cd client
npm run dev
```

### שלב 2: פתח את האפליקציה
פתח דפדפן והיכנס ל: http://localhost:3001

### שלב 3: התחבר או הירשם
- אם אין לך משתמש, לחץ "הירשם"
- אם יש לך משתמש, התחבר

### שלב 4: נסה את התרגום האוטומטי

1. **לחץ על "הוסף מילה"** בלוח הבקרה

2. **נסה מילים מהמילון המקומי** (תרגום מיידי):
   - הקלד: `hello`
   - תראה מיד: `שלום` ✅
   
   - הקלד: `cat`
   - תראה מיד: `חתול` ✅
   
   - הקלד: `beautiful`
   - תראה מיד: `יפה` ✅

3. **נסה מילה שלא במילון** (תרגום דרך API):
   - הקלד: `elephant`
   - המתן 0.8 שניות
   - תראה: `פיל` (מתרגם דרך LibreTranslate) ✅

4. **ערוך את התרגום**:
   - אפילו אם התרגום הופיע אוטומטית
   - אתה יכול לערוך אותו ידנית
   - לחץ "הוסף" כדי לשמור

## ⚡ פיצ'רים נוספים

### Debouncing
- המערכת ממתינה 0.8 שניות לאחר שהפסקת להקליד
- זה מונע קריאות מיותרות ל-API
- נסה להקליד `hello` אות אחר אות - תראה שהתרגום מתבצע רק בסוף

### Loading State
- בזמן התרגום תראה: **⏳ מתרגם...**
- זה מראה שהמערכת עובדת

### עריכה ידנית
- אם התרגום לא מדויק, פשוט ערוך אותו
- לחץ על שדה התרגום וכתוב מה שאתה רוצה

## 🐛 פתרון בעיות

### התרגום לא עובד?
1. בדוק שהשרת פועל (http://localhost:5000/api/health)
2. בדוק את הקונסול בדפדפן (F12)
3. נסה מילה פשוטה מהמילון: `cat`, `dog`, `hello`

### התרגום איטי?
- זה נורמלי למילים שלא במילון המקומי
- LibreTranslate API יכול לקחת כמה שניות
- מילים מהמילון המקומי מתורגמות מיידית

### התרגום לא מדויק?
- ערוך אותו ידנית
- שמור את התרגום הנכון
- בפעם הבאה תשתמש בתרגום השמור שלך

## 🎓 טיפים

1. **השתמש במילים פשוטות תחילה** - בדוק עם מילים מהמילון
2. **המתן לתרגום** - אל תלחץ "הוסף" מהר מדי
3. **ערוך אם צריך** - התרגום האוטומטי הוא כלי עזר, לא מושלם
4. **הוסף למילון** - אם משתמש במילים מסוימות הרבה, הוסף אותן למילון המקומי
