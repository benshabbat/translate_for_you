# 🚀 הוראות הפעלה מהירה

## לפני שמתחילים
ודא שיש לך MongoDB מותקן ופועל במחשב שלך, או השתמש ב-MongoDB Atlas.

## שלב 1: הגדרות Backend

1. פתח טרמינל והיכנס לתיקיית השרת:
```powershell
cd server
```

2. ערוך את הקובץ `.env` (הוא כבר קיים):
   - פתח את `server\.env`
   - שנה את `JWT_SECRET` למשהו אקראי ובטוח
   - אם אתה משתמש ב-MongoDB Atlas, שנה את `MONGODB_URI` לכתובת שלך

3. הרץ את השרת:
```powershell
npm run dev
```

השרת אמור לעלות על http://localhost:5000
אתה אמור לראות הודעה: "✅ Connected to MongoDB" ו-"🚀 Server is running on port 5000"

## שלב 2: הפעלת Frontend

1. פתח טרמינל **נוסף** (השאר את הטרמינל הקודם פתוח!)

2. היכנס לתיקיית הלקוח:
```powershell
cd client
```

3. הרץ את האפליקציה:
```powershell
npm run dev
```

האפליקציה תעלה על http://localhost:3000

## שלב 3: שימוש באפליקציה

1. פתח דפדפן והיכנס ל-http://localhost:3000
2. לחץ על "הירשם כאן" כדי ליצור משתמש חדש
3. התחבר עם המשתמש שיצרת
4. התחל להוסיף מילים ולתרגל!

## ⚠️ פתרון בעיות נפוצות

### MongoDB לא מתחבר
- ודא ש-MongoDB מותקן ופועל
- הרץ את MongoDB Compass או MongoDB Service
- או השתמש ב-MongoDB Atlas (בענן) ועדכן את `MONGODB_URI` ב-`.env`

### השרת לא עולה
- ודא שאין תהליך אחר שמשתמש בפורט 5000
- נסה לשנות את הפורט ב-`.env`: `PORT=5001`

### הפרונטאנד לא עולה
- ודא שאין תהליך אחר שמשתמש בפורט 3000
- נסה לשנות את הפורט ב-`vite.config.ts`

### בעיות בהרשמה/התחברות
- בדוק שהשרת פועל (http://localhost:5000/api/health)
- בדוק את הקונסול לשגיאות

## 📝 הערות
- כדי לעצור את השרת או הפרונטאנד, לחץ `Ctrl+C` בטרמינל
- כל שינוי בקוד יטען אוטומטית (hot reload)
- נתוני המשתמשים והמילים נשמרים ב-MongoDB
