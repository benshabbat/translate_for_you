# ✅ Checklist - רשימת בדיקות לפני הפעלה

השתמש ברשימה זו כדי לוודא שהכל מוכן להפעלה.

---

## 📋 הכנה ראשונית

- [ ] Node.js מותקן (גרסה 18+)
  ```powershell
  node --version
  ```

- [ ] MongoDB מותקן ופועל
  - [ ] מקומי: MongoDB Community Server מותקן
  - [ ] או: חשבון MongoDB Atlas נוצר

- [ ] npm מותקן
  ```powershell
  npm --version
  ```

---

## 🔧 התקנה

### Backend
- [ ] נכנסתי לתיקיית server
  ```powershell
  cd server
  ```

- [ ] הרצתי npm install
  ```powershell
  npm install
  ```

- [ ] קובץ .env קיים בתיקיית server
  - [ ] יש קובץ `.env`
  - [ ] שיניתי את `JWT_SECRET` למשהו אקראי
  - [ ] `MONGODB_URI` מוגדר נכון

### Frontend
- [ ] נכנסתי לתיקיית client
  ```powershell
  cd client
  ```

- [ ] הרצתי npm install
  ```powershell
  npm install
  ```

---

## 🚀 בדיקת הפעלה

### MongoDB
- [ ] MongoDB פועל
  ```powershell
  # בדוק שהשירות פועל
  Get-Service MongoDB
  ```
  או
  ```powershell
  # בדוק את התהליך
  Get-Process mongod
  ```

### Backend Server
- [ ] הפעלתי את השרת
  ```powershell
  cd server
  npm run dev
  ```

- [ ] השרת עולה ללא שגיאות
- [ ] רואה הודעה: "✅ Connected to MongoDB"
- [ ] רואה הודעה: "🚀 Server is running on port 5000"

- [ ] בדיקת health check
  - [ ] פתחתי דפדפן
  - [ ] נכנסתי ל-http://localhost:5000/api/health
  - [ ] רואה: `{"status":"ok","message":"Server is running"}`

### Frontend Client
- [ ] הפעלתי את הקליינט (בטרמינל נפרד!)
  ```powershell
  cd client
  npm run dev
  ```

- [ ] הקליינט עולה ללא שגיאות
- [ ] רואה הודעה עם הכתובת (לרוב http://localhost:3000)

- [ ] בדיקת UI
  - [ ] פתחתי דפדפן
  - [ ] נכנסתי ל-http://localhost:3000
  - [ ] רואה את עמוד ההתחברות

---

## 🧪 בדיקת פונקציונליות

### הרשמה והתחברות
- [ ] לחצתי על "הירשם כאן"
- [ ] יצרתי משתמש דמו:
  - שם משתמש: `test`
  - סיסמה: `123456`
- [ ] ההרשמה עברה בהצלחה
- [ ] הועברתי ללוח הבקרה

### הוספת מילים
- [ ] בלוח הבקרה, לחצתי "+ הוסף מילה"
- [ ] הוספתי מילה:
  - אנגלית: `hello`
  - עברית: `שלום`
- [ ] המילה התוספה בהצלחה
- [ ] המילה מופיעה ברשימה

### תרגול
- [ ] הוספתי לפחות 3 מילים
- [ ] לחצתי "התחל תרגול"
- [ ] רואה מילה באנגלית
- [ ] לחצתי "הצג תשובה"
- [ ] רואה תרגום בעברית
- [ ] סימנתי "ידעתי" או "לא ידעתי"
- [ ] עברתי למילה הבאה

### סטטיסטיקות
- [ ] בלוח הבקרה רואה:
  - [ ] סה"כ מילים (מעל 0)
  - [ ] מילים שתורגלו
  - [ ] אחוז הצלחה

### מחיקה
- [ ] לחצתי "מחק" ליד מילה
- [ ] אישרתי מחיקה
- [ ] המילה נמחקה

### התנתקות
- [ ] לחצתי "התנתק"
- [ ] הועברתי לעמוד ההתחברות

### התחברות חוזרת
- [ ] התחברתי שוב עם אותו משתמש
- [ ] המילים שלי עדיין קיימות
