# 🌐 תכונת תרגום אוטומטי

## ✨ מה זה?

כשאתה מקליד מילה באנגלית בשדה "הוסף מילה", המערכת תתרגם אותה אוטומטית לעברית!

---

## 🚀 איך זה עובד?

### 1. מילון מקומי (מהיר!)
המערכת כוללת מילון מקומי של מעל **120 מילים נפוצות** שמתורגמות מיידית:
- מילים בסיסיות: hello, world, cat, dog...
- מספרים: one, two, three...
- צבעים: red, blue, green...
- פעלים: eat, drink, sleep...
- ועוד...

### 2. API חיצוני (לכל השאר)
אם המילה לא נמצאת במילון המקומי, המערכת מנסה להשתמש ב-**LibreTranslate API** (חינמי ופתוח).

### 3. תרגום ידני
אם התרגום לא נמצא, אתה יכול להזין אותו ידנית.

---

## 📖 איך להשתמש?

1. **לחץ על "הוסף מילה"** בלוח הבקרה
2. **התחל להקליד** מילה באנגלית
3. **המתן 0.8 שניות** - התרגום יופיע אוטומטית
4. **ערוך את התרגום** אם צריך (ניתן לשנות)
5. **לחץ "הוסף"** כדי לשמור

---

## 💡 דוגמאות

| הקלד באנגלית | תקבל בעברית |
|--------------|-------------|
| `hello` | שלום |
| `cat` | חתול |
| `beautiful` | יפה |
| `computer` | מחשב |

## ⚙️ תכונות נוספות

### Debouncing
המערכת ממתינה 0.8 שניות לאחר שסיימת להקליד לפני שהיא מתרגמת.
זה חוסך קריאות מיותרות ל-API.

### אינדיקטור טעינה
כשהתרגום בתהליך, תראה: **⏳ מתרגם...**

### ערוך תרגום
גם אם המערכת תרגמה אוטומטית, אתה יכול לערוך ולשנות את התרגום.

### תמיכה offline
המילון המקומי עובד גם בלי חיבור לאינטרנט!

## 🎓 עבור מורים

### הוספת מילים למילון
אתה יכול להוסיף מילים נוספות למילון המקומי!

ערוך את הקובץ:
```
server/src/routes/translate.ts
```

הוסף מילים לאובייקט `dictionary`:
```typescript
const dictionary: { [key: string]: string } = {
  // ... מילים קיימות
  'apple': 'תפוח',
  'banana': 'בננה',
  // ... מילים נוספות שלך
};
```

### שיתוף רשימות
כל תלמיד יכול להוסיף מילים עם תרגומים אישיים.
המערכת רק עוזרת עם תרגום ראשוני.

## 🔒 פרטיות ואבטחה

- **המילון המקומי**: פועל על השרת שלך, אין שיתוף מידע
- **LibreTranslate API**: שירות חינמי ופתוח, לא דורש מפתח API
- **אין שמירה**: התרגומים לא נשמרים אלא אם אתה לוחץ "הוסף"

---

## ⚠️ מגבלות

### LibreTranslate API
- שירות חינמי ציבורי
- יכול להיות עמוס לפעמים
- לא תמיד מושלם לעברית
- אם לא עובד - המילון המקומי עדיין יעבוד

### פתרון
אם הAPI החיצוני לא עובד:
1. השתמש במילון המקומי (מילים נפוצות)
2. הזן תרגום ידנית
3. או הוסף את המילה למילון המקומי
